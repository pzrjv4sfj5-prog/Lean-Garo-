## DUPLICATE / OVERLAP ANALYSIS

### Grammar rules with IDENTICAL claims across sources (no conflicts):
These are safe to use directly тАФ fully validated:
- Word order SOV, verb final
- Past tense `-aha`
- Present `-a`
- Perfect `-jok`
- Future `-gen` (A'chik) / `-noa` (Mandi)
- Imperative `-bo`
- Negative imperative `-na-be`
- Accusative `-ko`
- Genitive `-ni`
- Question marker `-ma`
- Progressive `-ing-`/`-eng-`
- Classifiers: `mang-`, `sak-`, `king-`

### Grammar rules with DIALECT VARIANTS (not conflicts тАФ both valid):
| Feature | A'chik form | Mandi form |
|---|---|---|
| Progressive | `-eng-` | `-ing-` |
| Future | `-gen` | `-noa` / `-no-a` |
| Causative | `-at-` | `-et-`/`-it-` |
| Immediate future | `-gin-ok` | `-na-jok` |

### GENUINE CONFLICTS requiring native speaker resolution:

| # | Issue | Source A says | Source B says | Recommendation |
|---|---|---|---|---|
| 1 | Negation morpheme | S1: `-ja-` (academic standard) | S13: `-gija` (native confirmed) | **Keep both** тАФ `-gija` likely a contracted speech form. Implement `-ja-` algorithmically; keep `-gija` in corrections |
| 2 | `Ua` vs `Bia` for 3rd person | S1 uses `u-a`/`bi-a` interchangeably | S10 uses `Bia` consistently | **Both valid** тАФ `ua` = that/he/she (demonstrative origin); `bia` = he/she (more pronoun-like) |
| 3 | `Namengama?` vs `Namengaba` | S9 Wikipedia lists both | тАФ | **Both valid** тАФ question vs answer forms of same root |
| 4 | `O.am` (hello) | S11 blog only | S10 uses `Salam` | **Unverified** тАФ do not add to engine without native speaker confirmation |
