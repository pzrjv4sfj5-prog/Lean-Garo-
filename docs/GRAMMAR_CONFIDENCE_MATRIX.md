## RAKA RULE
| Rule | Confidence | Evidence |
|---|---|---|
| Raka is in root, never suffix | ✅ | Explicitly stated by Thangseng; verified across 20+ verb families |
| cha· root has raka | ✅ | All cha· forms confirmed with raka |
| kat root no raka | ✅ | `Anga kata` confirmed, `Kat·a` explicitly corrected |
| dak root has raka | ✅ | `Dak·a` confirmed, `dak·enga` corrected to match |
| ring root no raka | ✅ | `Ringa`, `ringbo`, `ringenga` all confirmed |
| tusi root no raka | ✅ | All tusi forms consistent |
| dong root no raka | ✅ | `donga`/`donggen`/`dongja` — explicitly confirmed by Thangseng (`Ango pen donga`) |
| re· root has raka | ✅ | All re· forms confirmed with raka |
| ka·saa root has raka | ✅ | `ka·saaha`, `ka·saenga`, `ka·saagen` all confirmed |
| nam root no raka | ✅ | `Nama`, `namja`, `namgija` all consistent |
| nang· root has raka | ✅ | Explicitly confirmed by Thangseng |
